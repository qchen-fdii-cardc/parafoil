## 动态规划

对于每个时间点，有四个可能的决策：

1. 向左打满，也就是 $\dot{\omega} = -\omega_{\max}$
2. 向右打满，也就是 $\dot{\omega} = \omega_{\max}$
3. 维持方向不变，也就是 $\dot{\omega} = 0$
4. 取一个介于 $-\omega_{\max}$ 和 $\omega_{\max}$ 之间的值

如果仅仅考虑前三种情况，那么这个问题就是一个动态规划问题。

目标状态：

$$
\begin{aligned}
x = 0 \\
y = 0 \\
\omega = -180^\circ
\end{aligned}
$$

### 动态规划问题的一般解法

1. 确定状态变量
2. 确定状态变量的取值范围
3. 确定决策变量
4. 确定决策变量的取值范围
5. 确定状态转移方程
6. 确定边界条件
7. 确定目标状态
8. 确定目标函数
9. 确定约束条件
10. 确定求解方法
